An extension for [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) to grep from [maple](https://github.com/liuchengxu/vim-clap/tree/master/crates/maple_cli).

## Get Started

```viml
Plug 'fannheyward/telescope-maple.nvim'

...
require('telescope').load_extension('maple')
```

## Usage

`:Telescope maple`
